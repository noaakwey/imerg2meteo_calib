## Итоговый алгоритм IMERG (Full-Distribution QM)
Для каждого сезона (DJF/MAM/JJA/SON), тренировочный период 2001–2015:
1. **Full-Distribution QM** (1000 квантилей, шаг 0.1%) — маппинг всего CDF, включая нули.
2. **Volume Scaling** — множитель `mean(station)/mean(IMERG_corr)` по тренировочному периоду.
3. Экстраполяция правого хвоста — линейная по наклону последних 5 квантилей.

## Итоговый алгоритм ERA5-Land (Moving Window QM)
Для каждого года *Y* и сезона:
1. Адаптивное окно обучения: `[max(1966, Y−15), min(2025, Y+15)]`.
2. **Full-Distribution QM** (1000 квантилей) + **Volume Scaling**.
3. При кросс-валидации: данные 2016–2021 исключаются из обучающего окна.

## Диагностика нестационарности ERA5-Land
Фиксированный период 1966–2015 даёт PBIAS ≈ −31% на валидации 2016–2021. Проверка по 10 станциям: сокращение периода до 2001–2015 снижает PBIAS до ±5%. Причина — климатический сдвиг: трансферная функция эпохи 1966–2000 не переносится на 2016–2021. Скользящее окно ±15 лет решает проблему и одновременно обеспечивает ретроспективный охват 1966–2025.
